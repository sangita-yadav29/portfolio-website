# Portfolio Website

A responsive personal portfolio website showcasing my skills, work experience, projects, and certifications as a Frontend Developer.

---

## 📌 About

This portfolio is built with pure **HTML5, CSS3, and JavaScript** — no frameworks, no build tools. It highlights my journey as a Frontend Developer, my internship experience at Cyfuture India Pvt. Ltd., and the real-world projects I've built using HTML, CSS, JavaScript, Bootstrap, and Python/Flask.

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Animated typing effect using [Typed.js](https://github.com/mattboldt/typed.js/)
- Dark-theme UI built with CSS Flexbox & Grid
- Sections: Home, About, Experience, Projects, Certificates, Contact
- Clickable certificate cards that open verified PDF certificates
- Downloadable resume (PDF)

## 🚀 Getting Started

1. Download or clone this repository
2. Make sure **all files stay in the same folder** (the HTML references them by relative path)
3. Open `index.html` directly in any browser — no server or build step required

```bash
git clone https://github.com/sangita-yadav29/portfolio.git
cd portfolio
# open index.html in your browser
```
